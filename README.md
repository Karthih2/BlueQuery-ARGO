# 🌊 BlueQuery

> *AI-Powered Ocean Intelligence Platform for ARGO Float Data*

BlueQuery is a cloud-native ocean analytics platform that allows users to explore complex ARGO oceanographic datasets using natural language queries instead of coding or manual NetCDF processing.

---

# ✨ Features

- 🤖 Natural Language → SQL Querying
- 🌡 Thermocline & Ocean Heat Content Analysis
- 📊 Interactive Charts & Scientific Visualizations
- 📡 ARGO Float Tracking & Mapping
- ☁️ AWS Cloud-Based Architecture
- ⚡ FastAPI + PostgreSQL Backend
- 🧠 Multi-LLM Support (Claude · Gemini · Groq)

# 🚀 Core Modules

### 1️⃣ Data Ingestion

Collects ARGO float NetCDF datasets from INCOIS, CSIRO, and Coriolis.

### 2️⃣ Data Processing

Processes temperature, salinity, depth, and timestamp data into structured formats.

### 3️⃣ Database Layer

Stores 5.6M+ observations inside PostgreSQL RDS.

### 4️⃣ AI Query Layer

Converts natural language questions into SQL using LLMs.

### 5️⃣ Visualization Layer

Generates scientific charts, maps, and ocean analytics.

### 6️⃣ Dashboard Interface

Interactive UI for querying and exploring ocean data.
